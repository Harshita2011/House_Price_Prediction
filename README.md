# Advanced Predictive Analytics – Lab Repository

This repository contains lab work for the **MDI3003 – Advanced Predictive Analytics** course.

## Contents

- **`LAB_01/`** – House Price Prediction using Linear Regression  
  Registration Number: **23MID0043**  
  An end-to-end regression study comparing linear, regularized, tree-based, and ensemble models on the Ames Housing, California Housing, and UCI Real Estate datasets.

## How to Use

Navigate to the relevant lab folder for notebooks, scripts, data, and the full technical report.

```bash
cd LAB_01
pip install -r requirements.txt
```
